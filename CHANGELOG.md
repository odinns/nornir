# Changelog

## 1.0.0 - 2026-05-04

First public release.

- Import personal archives into source-specific MySQL tables.
- Preserve intake records, import runs, artifacts, observations, and provenance links.
- Support working importers for ChatGPT, Facebook, X/Twitter, LinkedIn, Instagram, Gmail, Apple Messages, Apple Health, Wayback, media collections, and FidoNet.
- Build bounded source handoffs, Gmail evidence bundles, and search documents for review work.
- Publish the project site at `https://odinns.github.io/nornir/`.
